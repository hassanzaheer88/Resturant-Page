# Restaurant Page

A modern, responsive landing page for a restaurant built with HTML, CSS, and JavaScript.

## Overview

This project is a professional restaurant landing page designed to showcase the restaurant's menu, services, and contact information. The site is fully responsive and optimized for both desktop and mobile devices.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- (Optional) Node.js and npm for development

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hassanzaheer88/Resturant-Page.git
   cd Resturant-Page
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser, or
   - Use a local server for better development experience:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js http-server
     npx http-server
     ```

3. **Access the site**
   - Navigate to `http://localhost:8000` (or the port specified by your server)

## Customization

To customize this template for your restaurant:

1. **Update restaurant information** in `index.html`
2. **Replace images** in the `images/` folder
3. **Modify colors and fonts** in `css/style.css`
4. **Update menu items** in the menu section
5. **Add your contact details** and location information

## Deployment

The site is currently deployed on **Firebase Hosting**. To deploy your own version:

1. Install Firebase CLI:
   ```bash
   npm install -g firebase-tools
   ```

2. Login to Firebase:
   ```bash
   firebase login
   ```

3. Initialize Firebase in your project:
   ```bash
   firebase init
   ```

4. Deploy:
   ```bash
   firebase deploy
   ```
