# Restaurant Landing Page

A modern, responsive landing page for a restaurant built with HTML, CSS, and JavaScript.

## Overview

This project is a professional restaurant landing page designed to showcase the restaurant's menu, services, and contact information. The site is fully responsive and optimized for both desktop and mobile devices.

## Live Demo

🌐 **[Visit the Live Site](https://food-restaurent-site.web.app/)**

## Features

- ✨ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Modern UI** - Clean and attractive user interface
- 📱 **Mobile Optimized** - Touch-friendly navigation and layouts
- ⚡ **Fast Loading** - Optimized assets for quick page load times
- 🔗 **Easy Navigation** - Intuitive menu and navigation structure
- 📍 **Contact Information** - Easily accessible restaurant location and contact details
- 🍽️ **Menu Display** - Professional presentation of food offerings

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactive features and functionality
- **Firebase Hosting** - Deployment platform
